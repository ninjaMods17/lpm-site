Getting Started

Lynx Package Manager (lpm) v1.1.0 is a custom tool for managing Node.js dependencies with speed and simplicity.

Commands

  • lpm init

    Initialize a new project with package.json. Use -y or --yes for default values.

    [🐾] lpm v1.1.0
[✓] package.json created with default values!

    Example: lpm init -y

    Note: [🐾] and [✓] appear in green, [⚡] in cyan in the terminal.

  • lpm install

    Install all dependencies from package.json.

    [🐾] lpm v1.1.0
[⚡] Packages cached: 20% | ██       
[⚡] Installing axios@^1.9.0...
[⚡] Packages cached: 50% | █████    
[✓] Downloaded and extracted: axios@1.9.0
[⚡] Packages cached: 100% | ██████████
[✓] Dependencies installed in 0.8s

    Example: lpm install

    Note: [🐾] and [✓] appear in green, [⚡] in cyan in the terminal.

  • lpm install

    Install a specific package (e.g., axios) and add it to package.json.

    [🐾] lpm v1.1.0
[⚡] Packages cached: 20% | ██       
[⚡] Installing axios...
[⚡] Packages cached: 50% | █████    
[✓] Downloaded and extracted: axios@1.9.0
[⚡] Packages cached: 100% | ██████████
[✓] axios installed and added to package.json!

    Example: lpm install axios

    Note: [🐾] and [✓] appear in green, [⚡] in cyan in the terminal.

  • lpm start

    Run the "start" script defined in package.json.

    [🐾] lpm v1.1.0
[⚡] Running: node index.js
Hello, world! Running with lpm!
[✓] Script executed successfully!

    Example: lpm start

    Note: [🐾] and [✓] appear in green, [⚡] in cyan, [✗] in red in the terminal.

  • lpm help

    Display help information for all commands.

    [🐾] lpm v1.1.0
Lynx Package Manager (lpm) - Command Help
Usage: lpm  [options]
...

    Example: lpm help

    Note: [🐾] and [✓] appear in green, [⚡] in cyan in the terminal.

  • lpm -v, --version

    Display the version of Lynx Package Manager.

    1.1.0

    Example: lpm --version

    Note: Version appears in green in the terminal.

Install lpm v1.1.0

Follow these steps to get started with Lynx Package Manager:

[🐾] lpm v1.1.0
# Clone the repository
git clone https://github.com/ninjaMods17/lpm.git
cd lpm

# Link globally
npm link

# Test it
lpm --version
[✓] Version: 1.1.0

Note: [🐾] and [✓] appear in green, [⚡] in cyan in the terminal.
